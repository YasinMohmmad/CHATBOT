# 🤖 Smart NLP Chatbot

A Python-based rule-driven chatbot built using NLTK that interacts with users through natural language patterns.

This project was created to practice Python, NLP fundamentals, regex matching, and conversational design.

---

## 🚀 Features

✅ Greeting Detection
✅ Conversational Responses
✅ Name Recognition
✅ Built-in Calculator
✅ Time Response
✅ Joke Generator
✅ Typing Animation Effect
✅ Exit Commands

---

* Python
* NLTK
* Regex

## ⚙ Installation

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/CHATBOT.git
```

Install dependencies:

```bash
pip install nltk
```

Run:

```bash
python chatbot.py
```
